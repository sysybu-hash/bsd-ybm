# נתוני בדיקה מלאים - BSD-YBM

הקובץ `scripts/seed-test-data.mjs` מכין סביבת דמו מלאה שמתאימה לבדיקת CRM + ERP + סריקה רב-מנועית + מנויים + הגדרות + Meckano.

## הרצה

```powershell
npm run seed:test
```

הסקריפט עובד על הארגון הדמו בלבד לפי הדומיין:

```text
demo.bsd-ybm.test
```

הוא מנקה ומייצר מחדש את נתוני הדמו של הארגון הזה בלבד, כדי שאפשר יהיה להריץ אותו שוב ושוב בלי כפילויות.
